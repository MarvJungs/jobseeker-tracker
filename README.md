# Jobseeker Tracker
The current jobmarket, especially as a young professionell or recent university graduate, is getting really bad. Especially in current times where AI is easily able to replace tasks of a junior developer or similar, young jobseekers are struggling really hard to find a job. 
As i have been making these experiences myself I have decided to build this little tool which helps me to track my current progress on jobseeking while also expanding my skills and familarizing myself with different stacks.

## Tech-Stack
- Laravel (PHP/Backend)
- Bootstrap (CSS)
- MySQL (SQL)
- React (JavaScript/TypeScript/Frontend)
